理解Java 中的 transient 关键字

2018-05-25

transient 瞬时的、瞬变的、短暂的
meaning: working or staying somewhere for only a short time
Java 中的关键字 transient 表示,
在持久化 Serialization 中的特殊的对象数据成员我们不想用 Serialization 的机制来保存它的时候,
使用该关键字声明该字段是瞬时的.
