安装Mongodb

1. 下载最新版 MongoDB
2. 解压下载好的 taz 包, 并重命名
3. 编写配置文件 mongodb.conf
4. 创建数据库存放目录，并修改权限
5. 启动服务
6. 连接到 mongodb

请自行设置 $PATH
